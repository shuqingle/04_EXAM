# 第一章 信息化发展

## 1：新基建的3个方面

```
1 信息基础设施   
    网络基础设施：5G、物联网、工业互联网、卫星互联网
    新技术基础设施：人工智能、云计算、区块链
    算力基础设施：数据中心、智能计算中心
2 融合基础设施
	传统设施应用互联网、大数据、人工智能（智慧交通、智慧能源）
3 创新基础设施
	支持科研、技术开发、产品研制（公益属性）
	
```

## 2：数据价值化3个过程

```
数据资源化-数据资产化-数据资本化
```

## 3：数字经济的基础

```
数字产业化
```

## 4：智慧城市5个核心能力

```
1 数据治理（数据责权利管控）
2 数字孪生（物理空间到数字空间）
3 边际决策（执行端决策）
4 多元融合（社会关系和社会活动融合）
5 态势感知（模拟预测）
```

## 5：社会的智慧主体有哪些

```
自然人、手机账号汽车机器人
```

## 6：车联网3层体系

```
端：采集获取车信息，车的通信终端
管：车与路、人、网、车互联互通
云：云架构，海量信息汇聚
```

## 7：数字政府3内容

```
1 一网通办（一次登录，全网办理政务）
2 跨省通办（跨省申请，网办、代办、联办）
3 一网统管（强调治理，诉求事件实时发现解决）
```

## 8：产业数字化6特征

```
1 数字科技-变革生产工具
2 数据资源-关键生产要素
3 数字内容-重构产品结构
4 信息网络-市场配置纽带
5 服务平台-产业生态载体
6 数字善治-发展机制条件
```

## 9：智能制造成熟度5级模型

```
1级-规划级-基础条件规划
2级-规范级-单一业务
3级-集成级-跨业务
4级-优化级-数据挖掘、预测优化
5级-引领级-优化创新、新模式
```

## 10：产业数字化过程

```
1 数据-关键要素
2 价值释放-核心
3 数据赋能-主线
```

## 11：新型基础设施建设过程

```
1 技术创新-驱动
2 信息网络-基础
```

## 12：两化融合（信息化与工业化）的4个融合方面

